Overview¶

HPE Storage Array Exporter for Prometheus is a unified exporter that supports the HPE primary storage portfolio. The metrics provided differ by storage system type and release.

Delivery Vehicles¶

The exporter is delivered as:

  • Standalone binaries for Linux and Windows through GitHub releases.
  • Container images hosted on Quay, deployable via standalone Docker or via Kubernetes (with object definitions in YAML available in GitHub).
  • Helm charts hosted at Artifact Hub.
